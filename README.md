# Voice Assistant

A simple Python Voice Assistant that can recognize voice commands, respond using text-to-speech, and perform basic tasks such as telling the time, date, opening websites, and searching google.

## Features

* Voice recognition using SpeechRecognition
* Text-to-speech responses using pyttsx3
* Tell the current time
* Tell the current date
* Respond to greetings
* Tell jokes
* Open YouTube
* Open Google
* Open GitHub
* Search Google using voice commands
* Error handling for microphone and network issues

## Technologies Used

* Python
* SpeechRecognition
* PyAudio
* pyttsx3
* webbrowser
* datetime

## Installation

1. Clone the repository:

```bash
git clone https://github.com/shalinitiwari-25/Voice-Assistant.git
```

2. Move into the project folder:

```bash
cd Voice-Assistant
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

## How to Run

```bash
python main.py
```
